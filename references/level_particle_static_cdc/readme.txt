S.T.A.L.K.E.R. level.ps_static compiler/decompiler v.0.1

=========Использование============

Декомпиляция:	lpscdc.pl -d <input_file> [-out <outfile> -l <logfile> -cs]

-d <input_file> - входной файл (level.ps_static)
-o <outfile> - файл, куда распаковываем

Компиляция:	lpscdc.pl -c <input_file> [-out <outfile> -l <logfile> -cs]

-c <input_file> - распакованный файл
-o <outfile> - выходной файл

Общие опции:
-l <logfile> - файл лога.
-cs - формат файла ЧН или ЗП
